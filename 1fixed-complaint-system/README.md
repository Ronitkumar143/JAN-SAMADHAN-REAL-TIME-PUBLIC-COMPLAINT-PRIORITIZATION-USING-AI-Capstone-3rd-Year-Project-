# JanSamadhan – AI-Powered Grievance Portal

## Quick Start
```bash
pip install -r requirements.txt
python app.py
```
Open: http://127.0.0.1:5000

## Logins
- **Citizen**: Register with mobile number or email
- **Admin**: ID `admin@829206` | Password `123456`

## Features
- Citizen login/register (mobile or email)
- AI-predicted complaint category (ML model)
- Real-time status tracking (Pending / In Progress / Resolved)
- Admin dashboard: update status, add notes, view analytics
- AI cluster analysis of complaints
